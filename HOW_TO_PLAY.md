# ✦ 星图物语 · 玩法说明书

## 游戏简介

星图物语是一款以真实天文数据为底本的星座探索与叙事游戏。你将在逼真的星空中穿梭，点击亮星解锁古老的神话传说。

---

## 基本操作

| 操作 | 方式 |
|------|------|
| **旋转视角** | 鼠标拖拽（或手指滑动） |
| **缩放** | 滚轮缩放 |
| **点击星星** | 鼠标左键点击亮星 |
| **查看帮助** | 右上角 `?` 按钮 |

---

## 核心玩法

### 1. 探索星空
- 进入游戏后你会看到一片真实模拟的夜空
- **亮星**（带光晕的星星）是可交互的，鼠标移上会变手型
- 背景中数千颗暗星和银河带用于营造沉浸感

### 2. 收集星座
- 每颗亮星属于某个星座（如猎户座、大熊座）
- 点击星座中的星星，**按顺序**点亮
- 每点亮一颗星星，对应的连线会亮起
- 所有星星点亮 → 星座完成 ✅

> ⚠️ 如果点错了顺序，所有连线会重置，需要重新开始

### 3. 阅读神话
- 每次点击亮星都会弹出故事卡片
- 每颗星有三个版本的神话：
  - **希腊神话** — 奥林匹斯众神的故事
  - **中国星官** — 二十八宿与古代传说
  - **阿拉伯传说** — 沙漠中的星辰命名由来
- 卡片底部还有**科学知识**（距离、光谱型、深空天体）

### 4. 今夜可见
- 右上角「🌙 今夜可见」按钮
- 根据你的当前位置和时间，计算今夜能看到的星座
- 点击列表中的星座可直接定位过去

## 技术说明

- **星表来源**: Hipparcos 卫星数据（约 200 颗最亮星）
- **渲染引擎**: Three.js (WebGL) + UnrealBloomPass
- **天文计算**: 实时 RA/Dec 坐标投影，可见性基于经纬度/时间
- **所有星名使用拜耳命名法**（α Orionis、β Centauri 等）
